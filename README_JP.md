## 紹介

HyperOS専用の軽量なモジュールです! Xiaomi 12S ProとXiaomi Pad 6 Proに基づいて開発された自己使用なモジュールです。ほとんどの機能は他のモジュールに基づいており、オリジナルな機能はわずかです。
